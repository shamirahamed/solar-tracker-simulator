# Solar Tracker Simulator — AI Context

## Project
Single-axis solar tracker web simulator.  
Backend: Python FastAPI + pvlib.  
Frontend: HTML/CSS/JS, Chart.js, jsPDF.  
Deployed on Render.

## Structure

```
backend/app/
  main.py           FastAPI entry point
  models.py         Pydantic models
  api/routes.py     API endpoints
  services/
    solar.py        Solar position
    tracker.py      Tracker angles + fixed-panel POA irradiance
    shading.py      Shadow geometry + shading fraction
    shading_demo.py Full simulation loop

frontend/
  index.html
  styles.css
  app.js
```

## Simulation
- Resolution: 1 minute (1440 points/day)
- Solar position: pvlib solarposition
- Clear-sky: Ineichen model
- Tracker: pvlib singleaxis (Ideal / Limited / Backtracking)
- Fixed panel: latitude tilt, south-facing (north for southern hemisphere)
- POA irradiance: pvlib get_total_irradiance, Hay-Davies model
- Row shading: pvlib shaded_fraction1d
- GCR = panel_width / row_spacing

## Main API
POST /api/v1/simulate/day
