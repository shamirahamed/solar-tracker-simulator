About / Notes

Model & calculations

  • Solar position via pvlib get_solarposition.
  • Clear-sky irradiance: Ineichen model → POA via Hay-Davies transposition.
  • Tracker angle & backtracking: pvlib single-axis tracker with shaded_fraction1d.
  • Fixed panel baseline at latitude tilt, south-facing (north in southern hemisphere).
  • Irradiance comparison: Fixed vs No BT vs BT in W/m² and daily Wh/m².

How to read the visuals

  • Green panels in 2D view = single-axis tracker rows.
  • Red section = actual inter-row shading on the downstream panel.
  • Shadow line = displayed ground shadow (may be scaled; use shading % for accuracy).
  • Irradiance chart — three lines show daily energy gain of tracking vs fixed mount.
  • Click any chart to open fullscreen.

Legend

  • BT = Backtracking ON (pvlib backtrack=True)
  • No BT = Tracker without backtracking
  • Fixed = Fixed-tilt panel at latitude angle
  • Ideal = Unconstrained tracker angle
  • Limited = Angle capped at max rotation setting

UI features

  • 🌙 / ☀️ — Toggle light / dark theme.
  • Settings → Accent Color — Choose from 5 colour themes.
  • Device bar (bottom) — Preview mobile / tablet / desktop layout.
  • Download PDF — 4-page report with metrics and charts.
  • Clear-sky model only — no real weather data in this version.
