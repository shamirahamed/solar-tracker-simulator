About / Notes

Model notes

  • Solar position is calculated using pvlib.
  • Tracker angle and BT use pvlib single-axis tracking.
  • Irradiance / POA and power estimation follow a pvlib-based workflow.
  • Inter-row shading percentage uses pvlib row-shading logic where practical.

How to read the visuals

  • Black line = displayed ground shadow helper.
  • Red section = actual inter-row shading on the next panel.
  • Displayed shadow may be scaled / limited for readability.
  • Actual row-to-row impact is better understood from shading %.

Legend notes

  • BT = Backtracking
  • No BT = Without Backtracking
  • Ideal = unconstrained tracker angle
  • Limited = tracker limited by max rotation

Disclaimer

This is a tentative engineering simulation for practical comparison and visualization. Actual values may change with final design, site conditions, weather inputs, controls, losses, and validation.
